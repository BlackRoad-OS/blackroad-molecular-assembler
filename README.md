# BlackRoad Docs Site

Documentation platform for all BlackRoad products. Beautiful, searchable, versioned docs.

## Features

- **Multi-Product** - Unified docs for 400+ products
- **Search** - Instant full-text search
- **Versioning** - Docs for every release
- **API Reference** - Auto-generated from OpenAPI
- **Code Examples** - Runnable code snippets
- **Dark Mode** - BlackRoad design system

## Tech Stack

- Static generation for speed
- Algolia search
- MDX for interactive docs
- Auto-deploy on push
